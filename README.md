# 펫만나
📌 <b>프로젝트 개요</b>
<br>
<br>
지속 된 반려동물 보유 가구 증가로 인해 친구 만들기에 어려움을 겪는 사람들을 위한 홈페이지 입니다.
<br>
지역별 랜덤 매칭, 다양한 정보 공유, 반려동물의 새 친구를 만들어주는 것이 핵심입니다.
